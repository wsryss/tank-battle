# Tank Battle HTML

HTML坦克大战游戏

## 在线体验

下载后用浏览器打开 `index.html` 即可开始游戏。

## 操作说明

- **W / A / S / D** 或 **方向键**：移动坦克
- **空格** 或 **J**：射击
- **P**：暂停/继续

## 游戏规则

- 控制绿色坦克消灭所有红色敌方坦克
- 保护基地（黄色三角形），基地被摧毁则游戏结束
- 砖墙可以被子弹破坏，钢墙不可破坏
- 每关敌人数量递增
- 消灭敌人获得 100 分，过关奖励 500 分

## 技术说明

纯 HTML5 + Canvas + JavaScript 实现，无需任何依赖，单文件运行。
